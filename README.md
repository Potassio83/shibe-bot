# shibe-bot

A very simple (java) discord bot.

# Setup
1 run
```
java -jar Shibe-bot-1.0.jar

```
2 paste your token into token.txt 

and you are done!

# Commands
!shibe with this simple command the bot will send a random photo of a shibe
